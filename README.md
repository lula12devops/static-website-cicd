# Automated CI/CD Pipeline for Static website Deployment

## Overview
This project demonstrates the implementation of an automated CI/CD pipeline to deploy
a static website using modern DevOps tools and cloud infrastructure. 
The pipeline is designed to automatically build and deploy the application whenever changes are pushed to the repository

## Technologies Used
- HTML & CSS ( (static Webiste)
- Git & Github (version control)
- Docker (Containerization)
- Jenkins (CI/CD Automation)
- AWS EC2 (Cloud Infrastructure)
- Linux (Server Administration)

## Project Scope
- Source code versioning using Github
- CI/CD automation of a static website using Docker
- Deployment on cloud infrastructure hosted on AWS EC2

## Status 
Project infrastructure and CI/CD pipeline are currently being rebuilt and refined.
